I'd like to suggest these three songs for a TUG Xmas set - these are all kind of ukey xmas songs (to me at least) and they seem to fit together pretty well. 


Mele Kalikimaka - https://www.youtube.com/watch?v=FV_BGqgbxdc

Feliz Navidad - https://www.youtube.com/watch?v=xMtuVP8Mj4o
Christmas Island - https://www.youtube.com/watch?v=F7JUBN1YXaA
Attached is a first pass at song sheets in both PDF and TXT (chordpro) format for each - I've included an instrumental break in each - repeating the chord pattern in the verse/chorus. We need to figure out an intro and outro for each song. We'll no doubt make changes and we can revise the sheets as we go. 
Maybe we could play through a couple of these at the next meeting  - the first pass through should be pretty spooky.
