Here's the song list (so far) for April - June.  I've also attached the PDF of the songs sheets - this is the first pass - so keys, etc will change - but you can get a look at them - and each one has a You Tube link if you're not familiar with the tune. 

If you have songs you want to add - email them to me. If you can send the song in chord pro format, or I can find it in ChordPro, the song will get added pretty quickly -  if they need to be re-typed it can take a while.  

We'll do the first set of songs from this list on the April 9 meeting.
