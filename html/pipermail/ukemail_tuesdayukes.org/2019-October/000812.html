So, I've been thinking about the Hotel CA ending, and it seems like we can just repeat the intro twice. And, it turns out that the very end of the guitar duet can be mimicked pretty easily by just playing a 3 note arpeggio on the three highest strings on each chord. The 3 note pattern over a 4 beat measure emphasized different notes each time through. If two people do the same thing on different chord voicings, it sounds pretty good to me.
