Austin Parks and Rec approval our proposal to offer a Ukulele Intro Class at 6:30 each Tuesday.  

We're calling it the Tuesday Ukes Flash Class - we'll use the approach/materials we used at St John's Library and at Central Market. We'll bring the loaners - and we have a one-page 3-fold Flash Class pamphlet that we'll hand out - and we'll add a strumming pamphet and chord pamplet as they get done. The class will be just 30 minutes - then we'll transition into our regular TUG meeting. 

So, TUG will start as usual at 7:00 - if we have folks show up for the Flash Class at 6:30 that will take priority. If you want to show up at 6:30 - that's great - If no Flash Classers show up we'll use that time to work on other uke skills - like the tremolo strum or jazz chords - but the TUG meeting will still start at 7:00
