Here's what's up for the next TUG meeting Tuesday, July 20, 2019 Hancock Center

Attached is the weekly song PDF file (along with a .archive file for OnSong users).  Also attached is a version of Trouble in Mind with the chords (for G) in the first verse. Also the blues pattern shows the chords along with the roman numerals.
