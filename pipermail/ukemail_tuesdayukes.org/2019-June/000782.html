For anyone interested. 

I'm going to be doing a weekly singalong at the Westminster Manor nursing facility (The Arbors). We did a month of "classes" for the residents - and there was some interest - but these folks have multiple challenges. We were able to get some folks strumming ZED chords - and got some folks playing ostinado along with chords.   Residents don't show up, they come and go, they don't get what's going on - but some of them strum, and some of them sing, some do both.The activity director (Monica) is working on learning the ukulele so she'll be able to use that going forward. 

To support her and the residents I'd like to get a small group  of folks who could dedicate an hour each Monday to playing for/with these folks. I think if can get something resular going Monica will get more particiaption. 

The session is 10:30am each Monday at Westminster Manor (4100 Jackson, Austin) - we go for about 45 minutes. Attached is the song list we'll be working from. From that list we'll do six or seven songs each week. We repeat each song, and we play slowly - mostly using a simple downstrum.  We connect an iPad to a big screen TV to show the lyrics/chords, etc. 

Anyway - if you're interested email me 

Thanks

Kevin
